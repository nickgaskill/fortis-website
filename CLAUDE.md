# Fortis Home Solutions website

Public marketing site for Fortis Home Solutions, a construction & remodeling company in Central Texas (Austin, Georgetown, Lakeway), owned by Nick Gaskill. Live via GitHub Pages; every push to `main` republishes the site automatically in about a minute.

## Architecture

Plain static HTML/CSS/JS — no framework, no build step. Twenty-three real pages, each a full standalone page: `index`, `services` (an overview hub), the five service-category pages (`interior-remodels`, `exterior-living`, `home-additions-sunrooms`, `new-construction`, `other-services`), plus `gallery`, `process`, `reviews`, `contact`.

- **Navigation**: a sticky header with the wordmark (a small `logo-mark.svg` icon sits left of the "Fortis Home Solutions" text inside the `.logo` link; `.logo-mark` is sized at `1.5em` so it scales with the wordmark), a "Free consultation" button, and a single **"Menu" button** that click-toggles a **dropdown panel** (`.menu-panel`, `id="menuPanel"`) listing Home, Services (with Interior Remodels / Exterior Living / Home Additions & Sunrooms / New Construction / Other Services sub-links under `.menu-sub`, each linking to its own dedicated page), Gallery, Process, Reviews, Contact. `services.html` is an overview hub whose five cards (the last, `.svc-wide`, spans full width) link into those pages; the category pages share one layout (`page-hero` → intro + `.cat-list` of services + `.cat-note` → CTA band). **Interior Remodels** and **Exterior Living** instead show a clickable `.tile-grid` (dark-overlaid `.tile-ph` placeholder + white `.tile-corner` bracket + `.tile-label`) each linking to six sub-pages. Interior: `kitchen-remodels`, `bathroom-remodels`, `custom-cabinetry`, `painting`, `flooring`, `whole-home-renovations`. Exterior: `landscape-design`, `landscaping`, `hardscaping`, `pools`, `outdoor-living-spaces`, `sport-courts`. Sub-pages are reachable only from their parent's tiles (not the menu) and each has a “← Back to …” link. Swap `.tile-ph` for an `<img>` as real photos arrive. `site.js` toggles the panel's `hidden` attribute and closes it on outside-click, Escape, or link-click. The phone number is not in the header — visitors reach it via the "Free consultation" button and the contact page. Under 560px the "Free consultation" button hides so the wordmark and Menu fit.
- The **header and footer are copy-pasted into all twenty-three pages** — if you edit one (add a page link, change the phone number), make the same edit in every HTML file. The `aria-current="page"` marker is set on the current page's nav links.
- `styles.css` — all styling. The palette and fonts are CSS variables in `:root` at the top (`--navy` deep blue, `--gold`/`--gold-bright`, `--paper` warm cream, `--black`). Change a color there and it changes site-wide. The wordmark uses `--wordmark` (Cinzel), loaded from Google Fonts via a `<link>` in each page's `<head>` — the only external dependency. `section[id],[id].svc{scroll-margin-top}` keeps the sticky header from covering a heading when the Services dropdown jumps to an anchor.
- `site.js` — the Menu dropdown toggle (closes on outside-click, Escape, or link click) and scroll-reveal animations.
- The contact form posts to Formspree (`https://formspree.io/f/mpqgewbv`) and emails submissions to Nick.Gaskill@fortistexas.com.

## Conventions

- Brand voice: plain, direct, confident. Tagline: "Built different. Built Fortis." The home hero opens with the Latin meaning of the name — "for·tis — Latin adj. — strong; steadfast; built to last" (the old "From the slab to the last coat of paint" kicker is retired).
- Business facts: phone (512) 743-9690, email Nick.Gaskill@fortistexas.com, Instagram @fortistexas. The four customer reviews are verbatim quotes — don't reword them.
- Gallery projects use real locations/dates. Photo placeholders are `<div class="proj-ph">` blocks; replace with `<img>` tags as real photos are added (compress images to ~200-400 KB before committing).
- `index.html` carries the JSON-LD LocalBusiness schema and each page has its own meta description — keep these updated when business details change (they matter for Google).
- To preview locally, open any `.html` file directly in a browser — no server needed.
